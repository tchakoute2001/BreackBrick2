# Arkanoid Evolution

Arkanoid Evolution: arcade Java a 5 livelli dinamici. Include sistema di vite a cuori, mattoni indistruttibili e potenziamenti multi-ball (palline extra). Grafica 2D curata con effetti hover, audio immersivo e difficoltà crescente. Un mix di strategia e azione classica con gestione avanzata dei livelli e interfaccia utente moderna.
